### 🎨 Key Features

* **Fully Responsive Design:** Optimized for a seamless user experience across mobile, tablet, and desktop screens.
* **Modern UI/UX:** Clean typography, smooth transitions, and intuitive layouts built entirely from scratch.
* **Project Showcase:** A dedicated section displaying featured repositories with quick navigation links.
* **Contact Integration:** Structured connectivity forms for professional networking and freelance inquiries.

*Styling & Layouts: Custom CSS3 (Flexbox / CSS Grid / Media Queries)
*Interactivity: Modern Vanilla JavaScript (ES6+)
*Hosting: GitHub Pages (Automated CI/CD Deployment)
